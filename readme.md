# To see Project

## Windows ##

Creating venv
python -m venv .venv

to activate .venv <br> 
.venv/Scripts/activate

to install libraries <br>
pip install -r requirements.txt

## Mac Os ##

Creating venv
python3 -m venv .venv

to activate .venv <br> 
source .venv/bin/activate

to install libraries <br>
pip install -r requirements.txt
